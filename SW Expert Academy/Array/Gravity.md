# Gravity (삼성 Self Study Array 대표1)

![](./Gravity.JPG)

# 입력

첫번째줄에 test case의 수 T(1≤T≤100)가 주어진다. <br>
각 케이스의 첫째 줄 에 방의 가로 길이 N(2≤N≤100)과 방의 세로 길이 M(2≤M≤100)이 주어진다. <br>
다음 줄에는 N개의 상자들이 쌓여있는 높이H(0≤H≤M)가 주어진다.

# 입력 예시
```
1
9 8
7 4 2 0 0 6 0 7 0
```
# 출력
```
낙차가 가장 큰 값을 출력한다.
```
